# Atividade-de-Extensao
Este repositório contém as programações de um sistema de alimentador automático de suínos, desenvolvido com foco em automação industrial e controle de processos. O projeto integra controle de vazão de ração por helicoide e controle de locomoção com motores e encoders, utilizando estratégias como controle em malha fechada e controle em cascata.
